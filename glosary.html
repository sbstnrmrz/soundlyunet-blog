Glosario de Sistemas de Información

Sistema de Información (SI)

Un conjunto organizado de componentes interrelacionados (hardware, software, datos, personas y procesos) que recopilan, procesan, almacenan y distribuyen información para apoyar la toma de decisiones y el control en una organización.

Hardware

Se refiere a los componentes físicos y tangibles de una computadora o sistema informático, como el procesador (CPU), la memoria (RAM), el disco duro, el monitor, el teclado y los dispositivos de red.

Software

El conjunto de programas, instrucciones y reglas informáticas que permiten a la computadora realizar tareas específicas. Se divide principalmente en software de sistema (como Windows o Linux) y software de aplicación (como Microsoft Word o un navegador web).

Base de Datos

Una colección organizada de datos estructurados, almacenados electrónicamente en un sistema informático. Está diseñada para ser gestionada, actualizada y consultada de manera eficiente a través de un Sistema de Gestión de Bases de Datos (SGBD).

Red de Computadoras

Un conjunto de equipos (computadoras, servidores, periféricos) conectados entre sí mediante dispositivos físicos o inalámbricos, con el propósito de compartir recursos (como archivos o impresoras) e información.

Big Data

Se refiere a conjuntos de datos extremadamente grandes y complejos que los sistemas de software tradicionales no pueden capturar, gestionar ni procesar en un tiempo razonable. Se caracteriza por las "3 V": Volumen, Velocidad y Variedad.

Ciberseguridad

La práctica de defender las computadoras, servidores, dispositivos móviles, sistemas electrónicos, redes y datos de ataques maliciosos. Su objetivo es proteger la confidencialidad, integridad y disponibilidad de la información.
